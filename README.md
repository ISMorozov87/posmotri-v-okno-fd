# posmotri-v-okno-fd
Учебный проект: страница с подборкой видео и формой поиска.

## Цель проекта
Реализовать адаптивную страницу с сеткой контента, кастомным скроллбаром, стилизованной формой (чекбоксы, инпут, кнопка) и корректной доступностью (focus/focus-visible).
